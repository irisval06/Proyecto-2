# Proyecto-2
Análisis de comportamiento de clientes y segmentación para marketing
En este proyecto continué el análisis de datos para una empresa de comercio electrónico, profundizando en el estudio del comportamiento de los clientes a partir de un dataset previamente limpiado y estructurado.

El objetivo principal fue generar insights que permitieran optimizar campañas de marketing mediante una mejor segmentación de clientes y el análisis de sus patrones de consumo. Para ello, trabajé con datos más complejos, aplicando técnicas de análisis para identificar tendencias en las preferencias de compra, categorías de productos y niveles de gasto.

A lo largo del proyecto, realicé procesos de filtrado, agrupación y análisis de datos para explorar cómo varían los hábitos de consumo entre distintos perfiles de clientes. Esto permitió detectar segmentos clave basados en características como edad, comportamiento de compra y gasto acumulado, facilitando la identificación de oportunidades para personalización de campañas.

Además, se analizaron patrones temporales y relaciones entre variables para comprender mejor la evolución del comportamiento del cliente, proporcionando una base sólida para la toma de decisiones orientadas a negocio.

Como resultado, se generaron conclusiones accionables que ayudan a la empresa a dirigir sus estrategias de marketing de manera más eficiente y centrada en el cliente.

Este proyecto demuestra habilidades en análisis de datos, segmentación de clientes, uso de herramientas analíticas en Python y generación de insights orientados a negocio, consolidando la transición desde la preparación de datos hacia el análisis estratégico.
